## Run Localy

Clone project

```bash
git clone https://github.com/Joaoalencarl/disney-movie-api.git
```

Go to project and install

```bash
cd disney-movie-api
npm install disney-movie-api
```

Start

```bash
npm start
```
## Demonstration

for the demo we use the JSON file to define the API search specifications

```bash
{
    "filter":{
    "years": "2000,1995",
    "genre": "Comedy"
    }
}
```
